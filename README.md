## Setup
I use a Thinkpad for it's portability, keyboard, battery, and price. Any dev work is done by remoting into my home computer.

Thinkpad x220

8GB ram

120GB SSD

9-cell battery

Terminal - `termite`

Shell - `zsh`

Editor - `vim`

Window Manager - `i3-gaps`

Launcher - `rofi`

Music - `spotify`
